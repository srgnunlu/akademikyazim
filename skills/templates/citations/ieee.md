# IEEE Citation Guide / IEEE Atıf Kılavuzu

**Primary use / Birincil kullanım:** Engineering, Computer Science, Electrical Engineering, STEM

## In-Text Citation / Metin İçi Atıf

- Numbers in square brackets: [1], [2], [3]
- Multiple: [1], [2] or [1]–[3] for consecutive
- Narrative: "As shown by Smith [1]…"
- Numbers assigned in order of first appearance

## Reference List / Kaynakça

**Journal article:**
```
[1] A. B. Smith and C. D. Jones, "Title of article," IEEE Trans. Neural Netw., vol. 12, no. 3, pp. 45–67, Mar. 2020.
```

**Conference paper:**
```
[2] A. B. Smith, "Paper title," in Proc. IEEE ICASSP, Barcelona, Spain, 2020, pp. 100–104.
```

**Book:**
```
[3] A. B. Smith, Title of Book, 2nd ed. Publisher, 2020.
```

**Online source:**
```
[4] A. B. Smith, "Article title," *Site Name*, Mar. 15, 2020. [Online]. Available: https://... [Accessed: Apr. 1, 2020].
```

## Key Rules / Temel Kurallar

- Author initials before last name (reverse of APA/Harvard)
- Journal name abbreviated per IEEE standard list
- Month abbreviated (Jan., Feb., Mar., …)
- References numbered in citation order, not alphabetical
- [Online]. Available: and [Accessed: date] for web sources
