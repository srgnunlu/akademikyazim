# Vancouver Citation Guide / Vancouver Atıf Kılavuzu

**Primary use / Birincil kullanım:** Medicine, Health Sciences, Nursing, Pharmacy

## In-Text Citation / Metin İçi Atıf

- Superscript numbers or numbers in parentheses: Smith¹ or Smith (1)
- Multiple: (1,3,5) or (1-3) for consecutive
- Numbers assigned in order of first appearance
- Same number reused when same source cited again

## Reference List / Kaynakça

**Journal article (up to 6 authors):**
```
Smith AB, Jones CD, Brown EF. Title of article. J Med. 2020;12(3):45-67.
```

**Journal article (more than 6 authors):**
```
Smith AB, Jones CD, Brown EF, et al. Title of article. J Med. 2020;12(3):45-67.
```

**Book:**
```
Smith AB. Title of Book. 2nd ed. Publisher; 2020.
```

**Book chapter:**
```
Smith AB. Chapter title. In: Jones CD, editor. Book Title. Publisher; 2020. p. 45-67.
```

**Website:**
```
Smith AB. Article title [Internet]. Site Name; 2020 [cited 2020 Mar 15]. Available from: https://...
```

## Key Rules / Temel Kurallar

- No spaces in author initials (Smith AB not Smith A.B.)
- Journal names abbreviated per NLM catalog
- Volume(Issue):pages — semicolon before volume, colon before pages
- References numbered in citation order, not alphabetical
- DOI format: https://doi.org/... (when available)
