# OSCOLA Citation Guide / OSCOLA Atıf Kılavuzu

**Primary use / Birincil kullanım:** UK Law, Commonwealth Legal Writing, European Law

## Cases / Davalar

**UK:**
```
R v Smith [2020] UKSC 12
Jones v Brown [2019] 3 WLR 45
```

**EU Court:**
```
Case C-123/20 Smith v EU Commission [2021] ECR I-456
```

## Legislation / Mevzuat

**UK Act:**
```
Human Rights Act 1998
```

**Statutory Instrument:**
```
Civil Procedure Rules 1998, SI 1998/3132
```

## Secondary Sources / İkincil Kaynaklar

**Book:**
```
A Smith, Title of Book (Publisher 2020) 45.
```

**Journal article:**
```
A Smith, 'Article Title' (2020) 12 Journal Name 45.
```

**Chapter in edited book:**
```
A Smith, 'Chapter Title' in C Jones (ed), Book Title (Publisher 2020) 45.
```

## Key Rules / Temel Kurallar

- No full stop at end of footnotes
- Author first name initial only, no comma before title
- Year in square brackets for law reports that identify by year
- Pinpoint references (page/paragraph): add number at end with no preceding punctuation
- Bibliography optional in OSCOLA — footnotes are primary citation method
