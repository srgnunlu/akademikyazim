# Harvard Citation Guide / Harvard Atıf Kılavuzu

**Primary use / Birincil kullanım:** Business, Management, General academic writing

## In-Text Citation / Metin İçi Atıf

| Case | Format |
|------|--------|
| Single author | (Smith 2020) |
| Two authors | (Smith and Jones 2020) |
| Three or more | (Smith et al. 2020) |
| Direct quote | (Smith 2020, p. 45) |
| Narrative | Smith (2020) found… |

Note: No comma between author and year (unlike APA).

## Reference List / Kaynakça

**Journal article:**
```
Smith, AB and Jones, CD 2020, 'Title of article', Journal Name, vol. 12, no. 3, pp. 45–67.
```

**Book:**
```
Smith, AB 2020, Title of Book, 2nd edn, Publisher, Place.
```

**Book chapter:**
```
Smith, AB 2020, 'Chapter title', in CD Jones (ed.), Book Title, Publisher, Place, pp. 10–25.
```

**Website:**
```
Smith, AB 2020, Article title, Site Name, viewed 15 March 2020, <https://...>.
```

## Key Rules / Temel Kurallar

- Author surname + initials (no period between initials in some styles)
- Year immediately after author, no parentheses in reference list
- Single quotes around article/chapter titles
- Alphabetical order, no numbering
- "viewed" date required for web sources
