# APA 7 Citation Guide / APA 7 Atıf Kılavuzu

**Primary use / Birincil kullanım:** Social Sciences, Education, Business, Medicine

## In-Text Citation / Metin İçi Atıf

| Case | Format |
|------|--------|
| Single author | (Smith, 2020) |
| Two authors | (Smith & Jones, 2020) |
| Three or more | (Smith et al., 2020) |
| Direct quote | (Smith, 2020, p. 45) |
| Narrative | Smith (2020) found that… |

## Reference List / Kaynakça

**Journal article:**
```
Smith, A. B., & Jones, C. D. (2020). Title of article. Journal Name, 12(3), 45–67. https://doi.org/...
```

**Book:**
```
Smith, A. B. (2020). Title of book (2nd ed.). Publisher.
```

**Book chapter:**
```
Smith, A. B. (2020). Chapter title. In E. Editor (Ed.), Book title (pp. 10–25). Publisher.
```

**Website:**
```
Smith, A. B. (2020, March 15). Article title. Site Name. https://...
```

## Key Rules / Temel Kurallar

- Hanging indent for all references (0.5 inch)
- Alphabetical order by first author's last name
- DOI as hyperlink when available
- No "Retrieved from" unless URL is unstable
- Up to 20 authors listed before "…" ellipsis then last author
