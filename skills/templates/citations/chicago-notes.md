# Chicago Notes-Bibliography / Chicago Dipnot-Kaynakça

**Primary use / Birincil kullanım:** Law, Humanities, History, Theology

## Footnote / Dipnot Formatı

**First citation:**
```
1. Author First Last, *Book Title* (Place: Publisher, Year), page.
2. Author First Last, "Article Title," *Journal Name* vol, no. issue (Year): page.
```

**Subsequent citation (short form):**
```
3. Last, *Short Title*, page.
```

**Ibid (consecutive same source):**
```
4. Ibid., page.
```

## Bibliography / Kaynakça

```
Last, First. *Book Title*. Place: Publisher, Year.
Last, First. "Article Title." *Journal Name* vol, no. issue (Year): pages.
```

## Key Rules / Temel Kurallar

- Footnotes numbered consecutively throughout document
- Bibliography alphabetized by author last name
- Legal sources (legislation, court decisions) typically footnote-only, not in bibliography
- "Ibid." for consecutive identical source (same page: Ibid. / different page: Ibid., 45)
- Titles: *italics* for books/journals, "quotes" for articles/chapters

## Turkish Law Notes / Türk Hukuku Notları

Turkish legal citations generally follow footnote-only format. See [[law]] for TR-specific rules including Resmi Gazete, Yargıtay, and Danıştay citation formats.
