# STATUS.md Template

Copy to your project root as `STATUS.md`.
Auto-updated at every session end. Read at session start by `/tezatlas` and `/resume`.

---

```markdown
# STATUS.md — [Project Name / Proje Adı]
Last updated: [YYYY-MM-DD HH:MM]
Session count: [N]
Mode: [Workflow Assistant / Research Copilot]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 1. Project Snapshot

- Document type: [thesis / article / conference / lit-review / report / book-chapter / grant / research-proposal]
- Writing language: [TR / EN / DE / FR / ...]
- Research field: [field]
- Phase: [Phase Name — Phase Number]
- Phase completion: [X%]
- Days active: [N] (since [START_DATE])
- Total sessions: [N]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 2. Current Position

- Working on: [exact section — e.g., "Chapter 3, Section 3.2 — State Capacity Theory"]
- Last completed: [file + section title + word count]
- File path: [chapter_3_2.md]
- Last footnote: #[N]
- Total words: [N]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 3. Next 3 Actions

1. [IMMEDIATE — next thing to do when session opens]
2. [THIS SESSION — main goal]
3. [UPCOMING — what comes after]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 4. Open Questions

- [ ] [Question that needs resolution — e.g., "Which statistical test for H2?"]
- [ ] [Question 2]

(Delete entries when resolved)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 5. Blockers

- [ ] [Blocker — e.g., "Missing source: Acemoğlu_2005_Institutions.pdf — download needed"]
- [ ] [Blocker 2]

(Delete entries when resolved)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 6. Source Status

- Total in /sources/: [N]
- Read + noted: [N]
- Saturation: [YES / NO / PARTIAL]
- Deferred pool: [N sources — see READING_REPORT.md]
- Last source read: [author_year_title.pdf]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 7. Phase Gate Status

Current gate criteria:
- [ ] [Criterion 1]
- [ ] [Criterion 2]
- [ ] [Criterion 3]

Gate blocked on: [specific condition or CLEAR]
Advisor checkpoint: [DONE / PENDING / N/A]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 8. Writing Context (PBK — Next Session)

Last paragraph summary: [main claim in 1 sentence]
Terms used: [term1, term2, term3]
Open thread: [unresolved point to pick up]
Next paragraph task: [transition direction]
Argument Tracker: [N/M subclaims complete in current section]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 9. Session Notes

[Free text — anything important to remember]
[Advisor feedback, key decisions made, changed direction]
```
