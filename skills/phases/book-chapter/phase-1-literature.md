# Phase 1 — Narrow, Complementary Literature
# Aşama 1 — Dar, Tamamlayıcı Literatür

## Gate Rule / Geçiş Kuralı

**EN:** Literature for a book chapter should serve the chapter's argument in the book's context — not attempt to be a complete review of the sub-field. The alignment_brief.md approved in Phase 0 defines the boundary.

**TR:** Bir kitap bölümünün literatürü, alt alana tam bir inceleme yapmaya çalışmak yerine, kitabın bağlamında bölümün argümanına hizmet etmelidir. Aşama 0'da onaylanan alignment_brief.md sınırı tanımlar.

---

## 1. Literature Scope — Narrow and Purposeful / Literatür Kapsamı — Dar ve Amaçlı

**EN:** A book chapter literature engagement differs from a standalone paper. You are not surveying a field — you are positioning a specific argument. Ask for each source: does this help make the chapter's argument in the context of the book?

**TR:** Bir kitap bölümünün literatür bağlaşımı, bağımsız bir makaleden farklıdır. Bir alanı taramıyorsunuz — belirli bir argümanı konumlandırıyorsunuz. Her kaynak için sorun: bu, kitabın bağlamında bölümün argümanını oluşturmaya yardımcı oluyor mu?

**Inclusion test for sources / Kaynaklar için dahil etme testi:**
- Is it directly relevant to the chapter's specific argument? / Bölümün özgül argümanıyla doğrudan ilgili mi?
- Does it position the chapter within the existing scholarly conversation? / Bölümü mevcut akademik konuşmada konumlandırıyor mu?
- Is it cited in related chapters? (Coordinate to avoid all chapters citing the same 5 papers) / İlgili bölümlerde atıflandırılmış mı?

---

## 2. Overlap Check with Other Chapters / Diğer Bölümlerle Örtüşme Kontrolü

**EN:** Review the chapter map from alignment_brief.md. For each adjacent chapter:
- What foundational sources will they likely cite? Do not duplicate unnecessarily
- Is there a shared theoretical framework? If so, one chapter should introduce it; you cite that chapter
- Are there key scholars whose work spans multiple chapters? Coordinate with editor

**TR:** alignment_brief.md'deki bölüm haritasını gözden geçirin. Her bitişik bölüm için:
- Hangi temel sourcesı muhtemelen atıflandıracaklar? Gereksiz yere kopyalamayın
- Paylaşılan bir teorik çerçeve var mı? Varsa, bir bölüm onu tanıtmalı; siz o bölümü atıflandırın
- Birden fazla bölümü kapsayan kilit akademisyenler var mı? Editörle koordinasyon sağlayın

```
Overlap audit / Örtüşme denetimi:
Chapters with potentially overlapping sources / Potansiyel örtüşen kaynaklı bölümler:
Coordination needed with / Koordinasyon gerekli:
Shared bibliography agreed? / Paylaşılan kaynakça kararlaştırıldı mı?
```

---

## 3. Source Categories for Book Chapters / Kitap Bölümleri için Kaynak Kategorileri

**EN:** Book chapters typically engage with:

**TR:** Kitap bölümleri genellikle şunlarla bağlaşır:

| Category / Kategori | Purpose / Amaç |
|--------------------|----------------|
| Foundational texts / Temel metinler | Establish the scholarly conversation your chapter enters |
| Recent advances / Son gelişmeler | Show the chapter is current |
| Contrary views / Karşıt görüşler | Demonstrate awareness of debate |
| Empirical evidence / Ampirik kanıt | Support specific claims |
| Other chapters in volume / Ciltteki diğer bölümler | Explicit connective tissue |

---

## 4. Literature Position Statement / Literatür Konum İfadesi

**EN:** Before reading begins, write a brief position statement: where does this chapter's argument sit in relation to existing scholarship? This becomes the opening of the literature discussion in the chapter.

**TR:** Okumaya başlamadan önce kısa bir konum ifadesi yazın: bu bölümün argümanı mevcut bursiyerlere göre nerede duruyor? Bu, bölümdeki literatür tartışmasının başlangıcı olur.

```markdown
## Literature Position / Literatür Konumu

**Chapter's specific contribution / Bölümün özgül katkısı:**
[What does this chapter add that the existing literature lacks?]
[Bu bölüm mevcut literatürde eksik olan neyi ekliyor?]

**Primary scholarly conversation entered / Girilen birincil akademik konuşma:**

**Where this chapter agrees with existing scholarship / Mevcut bursiyerlere katıldığı yer:**

**Where this chapter departs or challenges / Ayrıldığı veya meydan okuduğu yer:**
```

---

## 5. Bibliography Format Coordination / Kaynakça Format Koordinasyonu

**EN:** Confirm citation format with the editor NOW. Changing citation format at revision stage is significant wasted work. Different publishers and series use different formats.

**TR:** Atıf biçimini ŞİMDİ editörle doğrulayın. Revizyon aşamasında atıf biçimini değiştirmek önemli boşa giden çalışmadır. Farklı yayıncılar ve seriler farklı biçimler kullanır.

```
Citation format / Atıf biçimi: [Chicago / APA / MLA / custom]
Footnotes or in-text? / Dipnot mu metin içi mi?
Reference list at chapter end or volume end? / Kaynakça bölüm sonu mu cilt sonu mu?
Any series-specific style guide? / Seriye özgü stil kılavuzu var mı?
```
