# Aşama 4 — Yazım ve Entegrasyon

## Gate Rule / Geçiş Kuralı

**EN:** chapter_argument_map.md must be finalized before writing begins. Writing an academic chapter without a complete argument map produces text that requires structural revision — the hardest and most time-consuming type of revision.

**TR:** Yazmaya başlamadan önce chapter_argument_map.md kesinleştirilmelidir. Tam argüman haritası olmadan akademik bölüm yazmak, yapısal revizyon gerektiren metin üretir — en zor ve en zaman alıcı revizyon türü.

## 1. Writing Order / Yazım Sırası

**EN:** Write sections in this order. Starting with the body sections (not the introduction) is more efficient — you cannot write a good introduction until you know exactly what the chapter argues.

**TR:** Bölümleri bu sırayla yazın. Giriş yerine gövde bölümleriyle başlamak daha verimlidir — bölümün tam olarak ne savunduğunu bilmeden iyi bir giriş yazamazsınız.

```
1. Body sections / Gövde bölümleri (in order, following argument map)
2. Conclusion / Sonuç
3. Introduction / Giriş
4. Footnotes / citations review / Dipnot / atıf incelemesi
5. Abstract (if required) / Özet (gerekiyorsa)
```

---

## 2. First and Last Paragraphs — Book Integration / İlk ve Son Paragraflar — Kitap Entegrasyonu

**EN:** These two paragraphs carry the chapter's connective tissue to the volume. They are not optional. They distinguish an edited volume chapter from a standalone paper.

**TR:** Bu iki paragraf, bölümün ciltle bağlayıcı dokusunu taşır. İsteğe bağlı değildir. Bunlar, editörlü cilt bölümünü bağımsız bir makaleden ayırt eder.

**First paragraph / İlk paragraf must:**
- Signal the chapter's place in the book's argument
- Reference what previous chapters have established (use chapter numbers/authors)
- State the chapter's own argument clearly

**Last paragraph / Son paragraf must:**
- Restate the chapter's argument and its contribution to the book's thesis
- If not the final chapter: gesture toward what follows
- If the final chapter: contribute to the volume's closing synthesis
- Do NOT end with "further research is needed" — this is a book chapter, not a journal article conclusion

## 3. Style Guide Compliance / Stil Kılavuzu Uyumu

**EN:** Follow the editor's style guide precisely. Common requirements that trip up contributors:

**TR:** Editörün stil kılavuzunu tam olarak takip edin. Katkı sahiplerinin tökezlediği yaygın gereksinimler:

## 4. Word Count Discipline / Kelime Sayısı Disiplini

**EN:** Editors do not have flexibility on word counts in edited volumes — total volume length is contracted with the publisher. A chapter that exceeds its allocation forces cuts elsewhere. Do not exceed the contracted word count.

**TR:** Editörler, editörlü ciltlerde kelime sayıları konusunda esnekliğe sahip değildir — toplam cilt uzunluğu yayıncıyla sözleşmelidir. Tahsisini aşan bir bölüm, başka yerlerde kesintilere zorunlu kılar. Sözleşmeli kelime sayısını aşmayın.

```
Contracted word count / Sözleşmeli kelime sayısı: ___
Current draft count / Mevcut taslak sayısı: ___
Difference / Fark: ___
```

## 5. Self-Editing Checklist Before Submission / Göndermeden Önce Öz-Düzenleme Kontrol Listesi

**EN:** Before sending the draft to the editor:

**TR:** Taslağı editöre göndermeden önce:

**Argument / Argüman:**
- [ ] Chapter's central argument stated clearly in introduction / Bölümün merkezi argümanı girişte açıkça ifade edildi
- [ ] Every section advances the argument — no section is purely descriptive / Her bölüm argümanı geliştiriyor — tamamen tanımlayıcı bölüm yok
- [ ] Counter-arguments addressed, not avoided / Karşı argümanlar ele alındı, kaçınılmadı

**Integration / Entegrasyon:**
- [ ] First paragraph connects to book's argument / İlk paragraf kitabın argümanıyla bağlanıyor
- [ ] Last paragraph connects to book's argument / Son paragraf kitabın argümanıyla bağlanıyor
- [ ] Cross-references to other chapters included where appropriate / Uygun yerlerde diğer bölümlere çapraz referanslar eklendi

**Technical / Teknik:**
- [ ] Word count within contracted limit / Kelime sayısı sözleşmeli sınır içinde
- [ ] Style guide followed throughout / Stil kılavuzu boyunca takip edildi
- [ ] All citations in correct format / Tüm atıflar doğru biçimde
- [ ] Iron Rules: every claim traceable / Demir Kurallar: her iddia izlenebilir

## 6. Editor Review Cycle / Editör İnceleme Döngüsü

**EN:** Edited volumes go through multiple review stages. Plan for:

**TR:** Editörlü ciltler birden fazla inceleme aşamasından geçer. Planlayın:

```
Draft submission / Taslak gönderimi → Editor review / Editör incelemesi
Revision 1 / Revizyon 1 → Editor + possibly peer review / Editör + muhtemelen hakemli
Final draft / Nihai taslak → Copyediting / Kopya düzenleme
Proofs / Provalar → Author corrections / Yazar düzeltmeleri
```
