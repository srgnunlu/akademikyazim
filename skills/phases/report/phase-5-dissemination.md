# Phase 5 — Dissemination / Faz 5 — Yaygınlaştırma

## Purpose / Amaç

**EN:** A research report is useless if it sits on a shelf. This phase translates the written document into actionable communication formats tailored for different stakeholders.

**TR:** Bir araştırma raporu rafta duruyorsa işe yaramaz. Bu faz, yazılı belgeyi farklı paydaşlar için uyarlanmış eyleme geçirilebilir iletişim formatlarına dönüştürür.

---

## Steps / Adımlar

**1. Stakeholder Mapping / Paydaş Haritalaması**
- Who needs to act on this report? (Policymakers, management, public, practitioners)
- What format do they consume? (1-pager, slide deck, social media)

**2. Develop the Communication Plan / İletişim Planı Geliştirme**
- Define the rollout timeline.
- Assign responsibilities for dissemination.

**3. Draft Derivative Outputs / Türev Çıktılar Tasarlama**
- **Executive Briefing (1-pager):** The absolute core findings and recommendations, designed for very busy decision-makers.
- **Presentation Deck:** Visual-heavy slides for briefings.
- **Press Release / Public Summary:** Jargon-free summary for media or general public.

## Completion Criteria / Tamamlanma Kriterleri

- [ ] Communication plan developed and target audiences identified.
- [ ] Executive briefing (1-pager) drafted.
- [ ] Presentation slides outlined.
