# Hukuk Disiplin Modülü / Law Discipline Module

Hukuk, diğer sosyal bilimlerden farklı bir epistemik yapıya sahiptir.
Kaynak hiyerarşisi normatiftir; metodoloji kural-uygulama mantığına dayanır;
argüman geçerliliği kanıta değil, normun yorumuna bağlıdır.

Bu modül Faz 0'da yüklenir ve tüm hukuki yazım fazlarında aktif kalır.

---

## 1. Türk Hukuku Kaynak Hiyerarşisi

Hukuki argümanda kaynak ağırlığı, kaynağın hiyerarşideki yerine göre belirlenir.
Üst norm alt normu geçersiz kılar.

```
1. Anayasa (1982)
   ↓
2. Uluslararası Antlaşmalar
   (AY md. 90: temel haklara ilişkin antlaşmalar kanunların üstündedir)
   ↓
3. Kanunlar (Türkiye Büyük Millet Meclisi)
   ↓
4. Cumhurbaşkanlığı Kararnameleri (2017 sonrası — yürütme yetkisi kapsamında)
   ↓
5. Yönetmelikler (Cumhurbaşkanlığı / Bakanlık)
   ↓
6. Tebliğler / Genelgeler / Yönergeler
```

**Önemli çekinceler:**
- Cumhurbaşkanlığı Kararnameleri temel hakları, siyasi hakları veya kanunla düzenlenmesi gereken alanları düzenleyemez (AY md. 104/17)
- Uluslararası antlaşmaların kanunlardan üstünlüğü yalnızca temel haklar alanındadır (AY md. 90/5)

## 2. Hukuki Araştırma Metodolojisi

### 2.1 Dogmatik (Hukuk Doktrini) Yöntemi

**De lege lata:** Mevcut hukuku olduğu gibi analiz etmek.
```
Yasa / İçtihat → Yorum → Sistematik yerleştirme → Uygulama
```

**De lege ferenda:** Hukukun nasıl olması gerektiğine ilişkin öneri.
```
Mevcut normun tespiti → Eksikliğin gösterilmesi → Reform önerisi + gerekçe
```

Tez ve makale sonuç bölümleri genellikle her iki düzeyi de içerir:
önce de lege lata analizi, ardından de lege ferenda öneriler.

### 2.2 Karşılaştırmalı Hukuk Yöntemi

Farklı hukuk sistemlerindeki düzenleme ve içtihatların karşılaştırılması.

**Karşılaştırma için temel sorular:**
1. Hangi hukuk ailesi? (Roma-Cermen / Anglosakson / İslam / karma)
2. Karşılaştırılan ülkenin kuralı ne amaçla benimsemiş?
3. Pratikte nasıl uygulanıyor? (law in books vs law in action)
4. Türk hukukuna aktarılabilir mi? Hangi uyarlamalar gerekir?

**Sık seçilen karşılaştırma ülkeleri TR hukuku için:**
Almanya, İsviçre (medeni hukuk), Fransa (idare hukuku), ABD/İngiltere (Anglo-Sakson), AB direktifleri (uyum araştırmaları)

### 2.3 Tarihsel Yöntem

Bir normun tarihsel gelişimini izlemek.
```
Orijinal metin → Yasama gerekçesi (meclis tutanakları) → Reform süreçleri → Güncel hâl
```

### 2.4 Ampirik Hukuk Araştırması

Nicel veya nitel yöntemler hukuki olgulara uygulanır.
Örnekler: mahkeme kararı istatistikleri, hâkim anketleri, içtihat analizi.
Bu yöntem → etik kurul onayı gerekebilir (bkz. [[research-ethics]]).

## 3. Hukuki Argüman Yapısı

### 3.1 Subsümsiyon (Kıyas) — Temel Hukuki Mantık

```
[NORM]       Belirli koşullar gerçekleşince belirli sonuç doğar.
[OLGU]       Somut olayda bu koşullar gerçekleşmiştir.
[SONUÇ]      O hâlde bu sonuç doğmuştur.
```

Akademik metinde subsümsiyon şu biçimde işler:
```
"X Kanunu'nun Y maddesi uyarınca [normatif koşul].
Somut olguda [olgu tespiti].
Dolayısıyla [hukuki sonuç]."
```

### 3.2 IRAC (İngilizce metinler için)

```
Issue    → Hukuki sorun nedir?
Rule     → Uygulanacak norm / içtihat nedir?
Analysis → Norm somut olguya nasıl uygulanır?
Conclusion → Hukuki sonuç nedir?
```

### 3.3 Yorum Yöntemleri

Bir normun anlamı tartışmalıysa yorum yöntemi açıkça belirtilmeli:

| Yorum Yöntemi | Açıklama | Kullanım |
|---------------|----------|---------|
| **Lafzi (Literal)** | Kelimelerin sözlük anlamı | Açık metin |
| **Sistematik** | Normun sistem içindeki yeri | Diğer maddelerle bağlam |
| **Tarihsel** | Kanun koyucunun kastı | Meclis tutanakları |
| **Amaçsal (Teleolojik)** | Normun amacı ve işlevi | Boşluk doldurmada |
| **Anayasal Yorum** | Anayasa'ya uygunluk | Her zaman kontrol et |

## 4. Mevzuat Atıf Formatları (Türk Hukuku)

### Kanun Atfı

```
Tam ilk atıf:
5411 sayılı Bankacılık Kanunu (RG. 01.11.2005, S. 25983), md. 3.

Madde atfı (sonraki):
Bankacılık Kanunu md. 56.
```

**Zorunlu bilgiler:** Kanun numarası + adı + Resmî Gazete tarihi ve sayısı + madde

**Cumhurbaşkanlığı Kararnamesi:**
```
Cumhurbaşkanlığı Teşkilatı Hakkında Cumhurbaşkanlığı Kararnamesi
(CBK No. 1, RG. 10.07.2018, S. 30474), md. 452.
```

**Yönetmelik:**
```
Bankaların İç Sistemleri ve İçsel Sermaye Yeterliliği Değerlendirme Süreci Hakkında
Yönetmelik (RG. 11.07.2014, S. 29057), md. 8.
```

## 5. İçtihat (Mahkeme Kararı) Atıf Formatları

### Yargıtay

```
Yargıtay [Daire No.] HD / CD, E. [Esas No.], K. [Karar No.], T. [Tarih].

Örnek:
Yargıtay 11. HD, E. 2021/3456, K. 2022/1234, T. 15.03.2022.
```

### Danıştay

```
Danıştay [Daire No.] D., E. [Esas No.], K. [Karar No.], T. [Tarih].

Örnek:
Danıştay 13. D., E. 2020/789, K. 2021/456, T. 10.06.2021.
```

### Anayasa Mahkemesi

```
AYM, [Esas No.], [Karar No.], T. [Tarih].

Bireysel başvuru:
AYM, B. No: [Başvuru No.], T. [Tarih], § [paragraf].

Örnekler:
AYM, 2019/40 E., 2020/35 K., T. 26.03.2020.
AYM, B. No: 2016/13494, T. 26.03.2020, § 45.
```

### Avrupa İnsan Hakları Mahkemesi (AİHM)

```
AİHM, [Dava Adı] / Türkiye, B. No: [Başvuru No.], T. [Tarih], § [paragraf].

Örnek:
AİHM, Kavala / Türkiye, B. No: 28749/18, T. 10.12.2019, § 212.
```

### ABAD (Avrupa Birliği Adalet Divanı)

```
ABAD, [Dava No.], [Taraflar], [Tarih], ECLI:[kod].

Örnek:
ABAD, C-131/12, Google Spain SL ve Google Inc. / AEPD, 13.05.2014,
ECLI:EU:C:2014:317.
```

## 6. Doktrin (Akademik Kaynak) Atıf Formatları

### Chicago Notes-Bibliography (TR hukuk standardı)

**Dipnot (ilk atıf):**
```
Kemal Oğuzman ve Özer Seliçi, Eşya Hukuku, 22. bs. (İstanbul: Filiz Kitabevi, 2021), 145.
```

**Dipnot (kısa atıf, sonraki):**
```
Oğuzman / Seliçi, 147.
```

**Kaynakçada:**
```
Oğuzman, Kemal ve Özer Seliçi. Eşya Hukuku. 22. bs. İstanbul: Filiz Kitabevi, 2021.
```

**Makale:**
```
Dipnot: Murat Doğan, "Kişisel Verilerin Korunması Hukukunun Temel Kavramları,"
Gazi Üniversitesi Hukuk Fakültesi Dergisi 22, s. 1 (2018): 34.

Kaynakça: Doğan, Murat. "Kişisel Verilerin Korunması Hukukunun Temel Kavramları."
Gazi Üniversitesi Hukuk Fakültesi Dergisi 22, s. 1 (2018): 23–58.
```

### OSCOLA (İngilizce hukuk metinleri)

```
Footnote: Kemal Oğuzman and Özer Seliçi, Property Law (22nd edn, Filiz 2021) 145.
Subsequent: Oğuzman and Seliçi (n 3) 147.
```

## 8. Türk Hukuku Tez Yapısı

### Kamu Hukuku Tezi (Standart)

```
Giriş
  - Konunun önemi ve sınırlandırılması
  - Araştırma sorusu ve hipotez
  - Yöntem ve kaynaklar
  - Bölüm planı

Bölüm I — Kavramsal ve Kuramsal Çerçeve
  - Temel kavramların tanımı
  - Tarihsel gelişim (gerekiyorsa)

Bölüm II — Mevcut Hukuki Düzenleme (De Lege Lata)
  - Mevzuat analizi
  - İçtihat analizi
  - Doktrin görüşleri

Bölüm III — Karşılaştırmalı Hukuk / Uygulama Sorunları
  - Seçilen ülke/ler ile karşılaştırma
  - Pratik uygulama sorunları ve örnek davalar

Sonuç ve Değerlendirme
  - Bulguların özeti
  - De lege ferenda öneriler

Kaynakça
  Mevzuat / Mevzuat (legislation first — Türk pratiği)
  İçtihat (case law)
  Doktrin (doctrine/books)
  Makaleler
  Diğer (raporlar, web kaynakları)

Ekler (gerekiyorsa)
```

### Özel Hukuk Tezi

Bölüm yapısı kamu hukukuna benzer; fark: II. Bölümde Borçlar Kanunu / Ticaret Kanunu merkezli analiz, İçtihat bölümünde Yargıtay ağırlıklı.

## 9. Hukuki Yazım Dil Kuralları

### Kaçın

```
❌ "Sanırım bu madde şunu demek istiyor..." (belirsizlik)
❌ "Muhtemelen mahkeme X diyecektir." (spekülasyon — kaynaksız)
❌ "Çok önemli bir hak ihlali söz konusudur." (dramatize)
❌ "Açıkça anlaşılmaktadır ki..." (açık değil, kanıtla)
❌ Kanun maddesini parafraz etmek — her zaman tam metni ver
```

### Kullan

```
✅ "X Kanunu'nun Y maddesi uyarınca... Bu düzenleme, doktrinde [Yazar, yıl]
   tarafından Z şeklinde yorumlanmıştır; Yargıtay ise W kararında..."
✅ Kesin dil: "geçersizdir", "hükümsüzdür", "ihlal oluşturmaktadır"
   (ama kaynaksız değil)
✅ Yorum yöntemini her belirsiz normda açık yaz:
   "lafzi yorumla...", "amaçsal yorumla..."
✅ Karşı görüşü belgele: "Aksi görüşte: [Yazar, yıl]"
```

### Latin Terimler (TR hukuk yazımında standart)

```
de lege lata      — mevcut hukuka göre
de lege ferenda   — olması gereken hukuka göre
in abstracto      — soyut olarak
in concreto       — somut olayda
ratio legis       — kanunun amacı / gerekçesi
contra legem      — kanuna aykırı
secundum legem    — kanuna uygun
ultra vires       — yetki aşımı
ex officio        — resen
```

## 10. Hukuk Yazımında Sık Yapılan Hatalar

- **Kanun metni yerine parafraz:** Madde metnini tam vermek yerine kendi cümlesiyle özetlemek — atlatmak zor, orijinal metni her zaman ver
- **İçtihadı doktrin gibi kullanmak:** Yargıtay kararı bağlayıcıdır ama yarı-bağlayıcı ya da yönlendirici düzeyde; derecesini belirt
- **Eski kararları tek kaynak yapmak:** Türk içtihadı hızlı değişir; kararın güncelliğini kontrol et
- **Karşı görüşü atlamak:** Hukuki doktrin çoğunlukla tartışmalıdır; hakim görüş + karşı görüş her ikisi de gösterilmeli
- **Kaynakçayı bölümsüz yazmak:** Mevzuat / İçtihat / Doktrin ayrı başlıklar hâlinde sunulmalı
- **RG bilgisi olmadan mevzuat atfı:** Resmî Gazete tarihi ve sayısı olmadan mevzuat atfı eksiktir
- **"Hukuk genel olarak şunu söyler" genellemesi:** Her iddia somut norma veya karara dayandırılmalı
- **De lege ferenda öneriyi kaynaksız yazmak:** Reform önerisi de karşılaştırmalı hukuk veya doktrinden desteklenmelidir

## 11. Hukuk için Özel Etik Uyarıları

- **Gizli dava dosyaları:** İzin alınmadan kullanılamaz
- **Anonimleştirme:** Yayımlanmamış kararlar veya davalar kullanılıyorsa taraf isimleri anonimleştirilmeli
- **Çıkar çatışması:** Avukat-araştırmacı için müvekkil bilgisine dayanan akademik çalışma ciddi etik sorun yaratır
- **KVKK + hukuki araştırma:** Kişisel dava verileri içeren ampirik çalışmalar KVKK kapsamındadır (bkz. [[research-ethics]])
